#  MyIdolo AI

MyIdolo AI is a simple web application that allows users to search and view mock statistics of popular X (Twitter) accounts in a clean dashboard interface.

---

## 🔥 Features
- Search X usernames (e.g. elonmusk, cristiano, nasa)
- View profile statistics (followers, following, posts)
- Trending idols quick access
- Save favorite idols locally (browser storage)
- Clean, responsive UI design
- Mock data system for demo purposes

---

## 🛠️ Built With
- HTML
- CSS
- JavaScript

---

## 🌐 Live Demo
https://my-idolo-ai.onrender.com

---

## 📦 Deployment
This project is hosted on Render and connected to GitHub for automatic updates on every commit.
